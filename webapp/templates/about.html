TfL Bike API
  • Source: Transport for London
  • Data: ~800 docking stations
  • Monitored: 21 stations within 800m of Imperial College
  • Interval: Every 60 seconds
  • Fields: Available bikes, e-bikes, empty docks, total capacity
  • Auth: No API key needed
Open-Meteo Weather
  • Source: Open-Meteo API
  • Location: 51.4988, -0.1749
  • Interval: Every 60 seconds
  • Fields: Temperature, humidity, precipitation, wind speed, WMO code
  • Auth: No API key needed
ESP32 + PIR Sensor
  • Hardware: ESP32 + HC-SR501
  • Function: Departure detection
  • Output: RGB LED + LCD 1602
  • Protocol: MQTT / HTTP
  • Purpose: Triggers real-time recommendation at the door
